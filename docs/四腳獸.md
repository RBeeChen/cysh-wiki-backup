**四腳獸**(英文:Gay　臺語:假(ㄍㄟˋ)　中文:基♂友　丹麥文:homoseksuel　阿拉伯文:مثلي الجنس　韓文:게이　高棉文:ខ្ទើយ　阿姆哈拉文:ጌይ　宿霧文:Bakla　寮文:ນໍ້າອະສຸຈິ　祖魯文:Izitabane　烏克蘭文:Гей　孟加拉文:গে　俄文:гей)

**四腳獸**是一種出現在**陽明學苑(學校宿舍)**二樓以上的人為產生的**基佬科多腳屬奇型種**生物，這種生物具有群體居住的習性，常常以**由複數個體結合在一起**型態出現，其中由兩個個體結合在一起的**四腳獸**型態最為常見，除此之外也有多種其他型態。這種生物生性十分內向，通常不會主動攻擊人類，但十分怕生如果被人類發現時候受到驚嚇，並試圖以**蕉♂流**的方式將該人類掰彎變成自己的同類，但由於期十分罕見也躲藏在宿舍暗處之中，所以至今還沒有人員受害的災情傳出。

**四腳獸**目前正面直接目擊這種生物全身的情報人存在(懷疑實際目擊過的人都被**蕉♂流**成同類了)，古老的文獻中紀載這種生物擁有四條人類雙腿而得名，目前對於這種生物的資料僅能從一些古老的文獻以及**某些特殊的美國摔♂角教學影片**中得知，並且還不清楚任何受害時的自保方式，目前僅能通過**不要在深夜到浴室去**的方式降低碰到**四腳獸**的機率。

也有一說，**四腳獸**也有可能為一隻**獨角獸**和另一個非獨角獸的生物的結合體，但在嘉中校園極為罕見，~~這種類型也不是只有嘉中會出現~~。這種**四腳獸**出沒地點高機率位於廁所。

根據傳言，據說**四腳獸**或**六腳獸**曾在[聖誕晚會](聖誕晚會 "聖誕晚會")於科學館出現。

也是根據傳言，某嘉中友校也曾經出現過此種生物。

## 生成條件

:   當有住宿男性慾火攻心且性傾向為同性戀時會有機率主動生成(若性傾向為異性戀則會生成另一種生物，詳見[「沐浴乳」](「沐浴乳」 "「沐浴乳」"))，通常有一個同性戀男性陷入這種狀態後會化身成**獨角獸**，由於奇群居的習性，會試圖尋找另一個**獨角獸**，如果尋找不到的話便會開始試圖通過**蕉♂流**的方式將身邊的正常人類掰彎變成自己的同類，並且在人煙稀少的隱僻處(大多在浴室)結合形成**四腳獸**，並試圖透過某種運動行為解除慾火攻心的狀態，在解除完狀態後又會變回人類繼續潛伏於人群之中。

## 不同型態

### 獨角獸

:   **獨角獸**是**四腳獸**還未結合群居之前的個體型態，由於牠明顯且堅挺的**獨角**而得名。

### 四腳獸

:   當兩隻**獨角獸**結合後形成的一種群居型態，也是目前最常見的型態。據說兩隻**獨角獸**在形成**四腳獸**的過程中會**發出基♂烈的摔♂角聲**，貌似是種爭奪群體間的主導地位的攻擊行為。

### 六腳獸

:   當三隻**獨角獸**結合後形成的一種群居型態，較為罕見，其結合原理相當於一隻**獨角獸**在與一隻**四腳獸**相結合。當群體中的個體數達到三個以上時，通常在群體最後方的為群體領袖，而最前方的為群體中地位最低者，中間的地位並無特別規率，所以當個體數達到三個以上時，**結合時的摔♂角聲會遠比四腳獸的結合摔♂角聲還要基♂烈很多**。

### 八腳獸

:   當四隻**獨角獸**結合後形成的一種群居型態，十分罕見，其結合原理相當於一隻**獨角獸**在與一隻**六腳獸**相結合。**八腳獸**是能出現在浴室中的最大群體型態，在接下來的更大的群體型態就無法出現在狹窄的浴室中。

### 人形蜈蚣

:   當五隻或以上的**獨角獸**結合後形成的一種群居型態，極其罕見，其結合原理目前未知，目前的物理學家推測**人形蜈蚣**的每一個個體間可能存在某種類似萬有引力的相互作用力使不同個體間緊密連接。**人形蜈蚣**由於其群體過於龐大而無法繼續存於浴室中，目前推測可能主要以出現在**門上掛了毛巾的寢室中**為主，所以若發現有人的寢室門上掛了毛巾，請盡速遠離該寢室的任何人員，以避免遭受**蕉♂流**，也有傳說深夜時走廊上會出現超過十餘個體形成的**人形蜈蚣**(?)。
