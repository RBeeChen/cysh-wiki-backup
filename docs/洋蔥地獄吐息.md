## 各項天賦&可習得技能介紹

### 在老師不注意時偷吃

:   正如字面所述，在做好料理後稱老師在回答問題、幫忙收拾爛攤子、清點餐具籃等不注意的時候偷吃食物。

:   :   此天賦可學會的技能:
    :   蔥(蒜)味吐息

#### 蔥(蒜)味吐息

:   :   這種技能只有在不小心加太多的蔥或蒜時使用，施術者之下料理後向前方深深吐氣，會造成周圍的人**食慾不振**、**噁心胃反**等症狀，同時在**半徑30公分的範圍內形成絕對領域**，所有人都會自動退出絕對領域的範圍。
    :   **注意:吐息系技能都是無法主動關閉的，所以使用開啟時請三思，若已開啟請遠離癡漢大道，除非你想就此被女性排擠。**

### 隨時偷吃

:   擺脫了老師是否注意的拘束(與自己的良心?)，隨時正大光明的偷吃食物。

:   :   此天賦可學會的技能:
    :   (無)

### 隨時隨地偷吃

:   需要點開"**跑到他組偷吃**"的支線天賦才可學習。連場地的拘束都擺脫了，自由穿梭於各組之間偷吃食物，需要有良好的人緣或極厚的臉皮才能辦到。(如果只有後者卻硬要點擊此天賦，請先向**聖馬爾定醫院**掛好急診。)

:   :   此天賦可學會的技能:
    :   (無)

### 做到一半偷吃

:   這個天賦大多用在做甜點的場合，例:偷吃餅乾糊等。但這天賦有些微的副作用，這會導致做出來的成品比別組少(大家都用就不會有這副作用了?)。

:   :   此天賦可學會的技能:
    :   (無)

### 還沒做就偷吃

:   這個天賦大多用在做甜點的場合偷吃原料，例:偷吃杏仁等。但這天賦有嚴重的副作用，由於所有食材都是按比例準備的，一開始就偷吃時才會導致破壞料理比例，可能會使**成品做不出來**或**成品味道詭異**(想使用之前可以提醒購買該食材的同學多買一點)。

:   :   此天賦可學會的技能:
    :   (無)

### 偷吃生的食材

:   擺脫只有做甜食時才能偷吃原料，就連做炒飯、麻婆豆腐等場合也可以偷吃原料，但需要點開支線天賦"**鐵胃銅舌**"才可學習(**強烈建議每一位幫家政課料理試吃評分的老師學習"鐵胃銅舌"此項支線天賦**)。學習這項天賦後甚至可以偷吃生蔥、生薑、生蒜、生洋蔥等地獄級食材，通常能使用這項天賦的人都具備**鐵胃**、**味覺感知異常**、**膽子特大**或**單純沒腦袋**等能力。

:   :   此天賦可學會的技能:
    :   生蔥地獄吐息、生蒜地獄吐息、洋蔥地獄吐息
    :   (生薑和辣椒是使不出地獄吐息的，強行使用只會單純讓施術者感到痛苦)

#### 生蔥地獄吐息

:   :   **蔥味吐息**的升級版，除了**殺傷力增強**與**"絕對領域"範圍提升至60公分"**外其他與**蔥味吐息**的效果差不多。

#### 生蒜地獄吐息

:   :   **蒜味吐息**的升級版，除了**殺傷力增強**與**"絕對領域"範圍提升至60公分"**並且**能對吸血鬼造成傷害**外其他與**蒜味吐息**的效果差不多。中醫師指出使用"**生蒜地獄吐息**"有**活通血脈**、**舒緩筋骨**、**加強心肺功能**、**降低"三高"**等好處。

#### 洋蔥地獄吐息

:   :   施術者啃食生洋蔥後向前方吐息，對前方30公分扇形範圍內的有機生命體**造成爆擊**，除了造成**食慾不振**、**噁心胃反**等症狀外，還有機會使人**短暫失去意識**和**嗅覺系統失靈**等狀態，且**具有半徑範圍一公尺的"絕對領域"**，**是目前所有吐息系技能中最強的存在**。

:   :   **注意:在施放此術的其間若經過癡漢大道就等著被女生公幹到畢業吧**

#### 食腐系列天賦

:   :   即為**"垃圾吃垃圾大"、"吃剩羹殘飯"與"吃廚餘桶"**三個支線天賦的統稱。

##### 垃圾吃垃圾大

:   :   食用製作過程中或裝盤過程中從容器裡掉出來的食物。為**食腐系列天賦**的基礎天賦。

##### 吃剩羹殘飯

:   :   食用他人吃過後剩下一兩口的剩菜、剩飯。需先學會**"垃圾吃垃圾大"**。

##### 吃廚餘桶

:   :   顧名思義，就是食用烹飪教室外的廚餘處理機或宿舍後面的藍色桶子中的食物。現今已無人使用。需先學會**"吃剩羹殘飯"**。

#### 腐食吐息

:   :   **吐息系技能**中已經失傳的傳說中的技能，施放方法未知，但可能與**"吃廚餘桶"**這項支線天賦的開啟方式有關。據說使用之後能**滅殺施術者眼前一切有機生命體**。
