**「沐浴乳」**(英文:Semen　臺語:洨(ㄒㄧㄠˊ)　中文:精液、嘉明　毛利文:Te rea　荷蘭文:sperma　阿拉伯文:المني　韓文:정액　法文:Sperme　高棉文:ទឹកកាម　阿姆哈拉文:ሴም　宿霧文:Semal　希臘文:Σπέρμα　寮文:ນໍ້າອະສຸຈິ　海地文:Espèm oswa po　祖魯文:Ubudoda　烏克蘭文:Семен　孟加拉文:বীর্য)

**「沐浴乳」**是一種出現在**陽明學苑(學校宿舍)**二樓以上的浴室中的一種**人為產生的史萊姆種族生物**，不具有任何行動能力，亦不具備有任何智商，但目前已確認是一種由多個生命個體聚集形成的有機生命體，其不會主動向人類進行攻擊，但當有人對它進行肢體碰觸時(大多是不小心踩到)，它本身詭異的濃稠度及滑溜溜的觸感會對該人類的心理造成嚴重的陰影面積，可能造成噁心不適、情緒激動、粗口連篇等多項症狀，且如果受害者接觸到的皮膚有傷口的話會有機率患上**B型肝炎**、**梅毒**、**愛滋**等疾病，由於被害者遭受攻擊後會在第一時間以**「這只是沐浴乳而已」**或**「我只是踩到沐浴乳罷了」**之類的想法來安慰自己試圖減小心理陰影面積而得名。

對付**「沐浴乳」**的唯一方法就是在進入浴室前仔細觀察是否有**「沐浴乳」**的存在，並用水將其驅趕至下水道中，~~或是將全宿舍的覽趴都剪掉~~。其常見的物理特徵是一團淡白色凝膠狀的液態生命體，但也有部分異種的存在，其濃稠度主要取決於生成者的生理狀況。據說在紫光燈下會產生些微螢光。

## 生成條件

:   當有住宿男性慾火攻心且性傾向為異性戀時會有機率主動生成(若性傾向為同性戀則會生成另一種生物，詳見[四腳獸](/index.php/%E5%9B%9B%E8%85%B3%E7%8D%B8 "四腳獸"))，生成者會到浴室經歷近乎虛脫的施力後將其生成，其中一名生成者像我們講述:「當你試過，你就會知道有個椅子讓你坐著是多麼舒服的事了」，大多數生成者發現**「沐浴乳」**的生成後會立即以水將其驅趕，並免其攻擊人類，但偶爾會出現生成者在生成過程中陷入意識茫然、欲仙欲死的精神狀態以至於無法將其驅趕，或是單純忘記，但也有可能是故意將其留下(不管是哪種都幹他媽的超級沒良心)，這樣就有可能發生前面所述的受害事件的發生。

:   有宗教學者認為這些**「沐浴乳」**生成者可能是受到七宗罪的**色慾**在現世中的化身--**車群女魔**(詳見 [日本文化研究社](/index.php/%E6%97%A5%E6%9C%AC%E6%96%87%E5%8C%96%E7%A0%94%E7%A9%B6%E7%A4%BE "日本文化研究社")中的傳奇人物)的蠱惑，或是已經被其調教成為的為其效命的邪教徒的作為，目前還不知道為什麼要進行對此種生物的生成，最有可能性的推測是，這是一種能助長**車群女魔**增長的一種邪教儀式，或是一種**女魔的攻擊手段**(?)

## 異種

### 水態**「沐浴乳」**

:   屬性:水。有「**可爾必思**」的別稱。當生成者生成**「沐浴乳」**的次數過於頻繁時就會產生的異種，其與正常種最明顯的差異就是**水態「沐浴乳」**幾乎沒有濃稠度可言，因此其殺傷力相比正常種還要小很多，有時甚至受害者只會感覺**今天的磁磚怎麼滑滑的**，並不會感覺到自己受到攻擊。**(但還是他媽的要給我沖掉啊)**

### 泛黃**「沐浴乳」**

:   屬性:雷。有「**維他露P**」的別稱。當生成者生成**「沐浴乳」**時的過程持續時間過長時會產生的異種，是**「沐浴乳」**過度吸收生成者的生命力的一種表現形態，其主要與正常種最明顯的不同就在於**泛黃「沐浴乳」**被肉眼所視時可以看到明顯的淡黃色甚至黃色，而不是普通的淡白色。其殺傷力比正常種還要強上兩倍，但也由於其顏色特徵較為明顯較容易在第一時間被發現並驅趕，所以比較不用擔心。**(但還是他媽的要給我沖掉啊)**

### 植物**「沐浴乳」**

:   屬性:草。有「**蔬菜汁**」的別稱。當生成者被**綠膿桿菌**感染時會生成的異種，其主要與正常種最明顯的不同就在於**植物「沐浴乳」**被肉眼所視時可以看到明顯的綠色，而不是普通的淡白色。其殺傷力十分強悍，除了能造成普通種五倍以上的心理陰影面積，且若被害者接觸部位有傷口的話還會有機率也受到**綠膿桿菌**感染，若不慎踩到建議盡速就醫檢查。但也由於其顏色特徵較為明顯較容易在第一時間被發現並驅趕，所以比較不用擔心。**(但還是他媽的要給我沖掉啊)**

:   目前專家學者們正在研究**植物「沐浴乳」**是否能行光合作用，試圖將其用於氧氣製造、改善地球環境等用途。

### 血腥**「沐浴乳」**

:   屬性:未知。有「**撒旦的洨**」的別稱。是種極其稀有的神奇寶貝，建議發現時立即將其收服。據說當生成者用力過猛時才會產生，由於其稀有性所以並沒有過多的墓籍資料進行考究，目前僅知其物理特徵上有肉眼可見的明顯紅色，且殺傷力是目前所知的**「沐浴乳」**當中最強的，經過專家學長的開會討論後一致建議，若沒有信心在第一時間將其收服，請撥打110請警政單位派出爆破物拆除隊並穿著防爆服進行驅逐。**(但還是他媽的要給我沖掉啊)**

## 生成案例

### 沒有水聲的反鎖浴室

:   目擊證人A指出，某天他前往浴室洗澡時發現**某個浴室沒有發出水聲**，於是A下意識的認為裡面沒有人，於是伸手試圖開門，發現們從內部被反鎖打不開後，又敲了敲門並詢問你面是否有人接沒有得到答覆後A認為「可能是門壞掉了吧」，由於當時處於洗澡的高峰期，所以沒有其他空浴室A只好先行回到寢室打遊戲，20分鐘過後，A又返回浴室，並發現原先被反鎖的浴室門是打開的狀態，於是高興地走入其中，當他踏入第一步後感覺自己的腳與浴室地面磁磚之間有種不明的液態黏稠物，並脫口而出一句:「媽的!幹!」

### 「乾」洗澡

:   我們派出的調查特工多次發現在半夜1點至3點左右的時間時，會出現有人兩手空空，並**沒有協帶任何毛巾、換洗衣物、臉盆、洗髮精等盥洗用品，僅協帶手機與入侵式耳機進入浴室**，且整個"洗澡"過程中除了沒有出現沖水的聲音，僅在最後結束的前5分鐘有出現一次以最強的水流從蓮蓬頭噴出撞擊地板磁磚及牆面的聲音，事後這些人走出浴室後的服裝與進入前一模一樣，具特工親口講述:「就連他們的**頭髮、身上的皮膚都像進入之前一樣乾**，若你真的要說有甚麼不同的話，唯一的不同就是他們的神情與臉部肌肉相比進去之前放鬆許多。」

:   在特工某次進行**「乾」洗澡**的觀察報告時，發現沒有聽到結束時會出現的水聲，從頭到尾都沒有出現任何聲音，人員離開後特工進入浴室中以紫光燈照射後發現地板、牆面甚至是天花板都出現淡淡的螢光反應，事後由**「沐浴乳」**學系的專家學者經現場採樣分析後，確定皆為**「沐浴乳」**。特工回憶時想我們表示:「當下的震驚比在公共廁所的天花板發現屎還要震驚十倍，要不是親眼看到我還以為RKI系列的A片都是假的呢(RKI系列的A片以使用道具屌拍攝誇張的射精畫面著名)。」
