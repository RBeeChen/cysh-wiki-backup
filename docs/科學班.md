# 辦學目的

(一)提供具科學潛能之優秀學生適性發展機會。  
(二)提供優越之教學環境及卓越師資，培養學生從事個別科學研究之能力及創造力。  
(三)培育兼具人文素養及科學專業知能之科學傑出人才，厚植國家競爭力。

# 簡介

2009年設立第一批科學班，辦理高中為建國中學、師大附中、竹科實中、臺中一中、臺南一中及高雄中學等6校。2011年核定第二批科學班，分別是武陵高中、彰化高中及嘉義高中等3校。2018年核定第三批科學班，為北一女中。  
一個歡樂可愛~~又無腦智障~~的班級，常常被其他班級錯誤認識，但事實上並非如此。  
剛入學的新生，其班級會暫時命名為018班，直到暑期輔導前，高一開學之前就會舉辦暑期輔導銜接課程（很常考試）。  
在暑輔時候會有迎新活動，是科學班必經之路
也是昔日嘉中唯三（音樂美術科學）有女生班級之一，只不過每年僅三名女生入學(2018年開始固定招收5名女性學生保障名額)，高三時仍需上第八節。  
有自己的社團[科學研究社](科學研究社 "科學研究社")。  
二年級下學期必須參加全國科學班聯合學科資格考，未通過者不得修讀三年級的個別科學研究，可能會面臨延畢

## 訪視

教育部長官會來學校詢問科學班課程、競賽、升學、研究的表現，若表現好，經費會比較多，新招的學生也會增加；反之，經費和招生名額會大砍。

### 訪視要點

* 有沒有預習與教師檢測預習的方式
* 有沒有用課本上
* 英文有沒有用全英文
* 有沒有充足的實驗課
* 入學考/資格考難度
* 個別科學研究進度
* 第二階段大學課程
* 研究倫理
* 在科學班的感想

## 課程

### 第一階段(高一高二)

* 數理科：兩年上完三年高中進度(還有部分舊課綱內容)
* 文科：正常進度與特殊班獨立進度，但是有加考。
* 英文科若有一定以上證照就可以免修
* 有別於現今108課綱
* 專題研究

### 第二階段(高三)

* 高中進度：複習學測+分科
* 預修大學課程
* 個別科學研究

# 升學表現

訪視委員不希望科學班學生去讀醫牙電資，而希望學生去讀基礎科學系，但是仍有大量科學班學生填志願時填醫牙電資。近年來嘉義高中頂尖的入學表現(尤其是台大醫科和台大電機)大多都是科學班學生締造的。  
2019年首度有女同學考取台灣大學醫學系(指考)  
2024年學測第11屆蘇姓考生應屆考取60級分，為嘉義地區之最，後來以個人申請榜首之姿考取台灣大學醫學系[[1]](#cite_note-1)  
當然也有優秀學生錄取國外知名大學(如林晉毅(2019畢業)錄取哈佛大學，嘉中史上第一，何承祐(2025畢業)錄取耶魯大學)
