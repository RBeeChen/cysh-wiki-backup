114學測聯合模擬試題​

1.傳聞老師參與出題，內容艱深

2.115學測生最後一次模考惡夢

教學特色

上課幽默風趣，筆記漂亮，並且會要求學生交筆記。偶爾會講一些話題，是個很好相處的人。教特殊班時會飆車並丟下頗難的數學問題。出考卷的時候喜歡丟下雄中普通班的超難題目、教師徵試的題目。(特殊班限定)
上課時，他總是帶著微笑，說話結束時會快速點頭，音調頻率偏高，講課時常常帶「ㄏㄚˋ｣字，偶爾聊廢話的時候會哈哈大笑。還有，老師是（非常）喜歡聽同學說八卦的。
可以中、日、英三語教學，但是日本學生來交流時才可以展現

加分機制​

  1. 抓數學老師的錯誤
  2. 講解題目(老師講到說要講解題目時，會的人可以寫黑板)

考卷​

  1. 身為雄中畢業生，老師喜歡找雄中的考古題來考科學班
  2. 對於科學班，曾經想過由半圓的不等式區域範圍，然後出了一個連GeoGebra做不出來的複雜圖形，還說它很像動感光波。
  3. 可以出簡單到不動筆就可以10秒內速解的題目，也可以出一些需要想半天的資優題目。
  4. 美術班的段考題會出畫圖題，要畫出上課所教過的內容
  5. 只要是是非題就會有很多陷阱，即使是科學班都很難十題全對
  6. 普通班卷難度甚高，每次出題都會請別的數學老師幫忙審題，如果沒有別的數學老師審題，難度會倍增！！！
  7. 對於"多選題只有一個選項是正確的"這件事有很大的熱誠
  8. 科學班加考會以大學段考的方式考試(全手寫，自己標題號，部份給分)，在最後一次加考中，潔勳老師出了11大題的考卷，範圍包括了矩陣、隨機變數、二項分布、幾何分布。考試時間140分鐘，滿分140，但是少有學生能寫完。

潔勳插值法​

在112學年度第一學期科學班第二次進階考試中，由牛頓插值法及拉格朗日插值法延伸的插值法。潔勳老師對於他出的情境題表示非常的自豪，受到學生們及老師們的讚賞。

112學年度下學期寒假作業考試​

潔勳老師第一次為全部的高一學生(還有高二學生)命題
難度高，一樣出了十題是非題，題目裡面滿滿都是陷阱，沒有仔細看的同學都會寫圈。
混合題需要比較多巧思，及跨觀念的運用整合能力才能拿下分數。

112學年度下學期高一期末考​

據說基本分42分。這是潔勳老師第一次在同一次段考出三份段考考卷(普通班、科學班、美術班)。

多選題中有一題只有一個選項

普通班範圍為排列組合機率期望值，潔勳老師稱此為高中學測範圍的惡夢。

在科學班的考卷之中，題目多到寫不完，但是潔勳老師自稱考卷難度比排列組合簡單，但是班平均59分。(範圍為三角函數（含圖形與複數極式）+ 數列與級數的極限)

113學年度上學期高二期末考(數A)​

1.有證明題請小心

2.難度跟當次國文科(汪楨玲老師命題)相當

3.難到考完出場整棟樓在唉

113學年度下學期高二第二次期中考(數A)​

下了「輕」手，所以有四個人考了幾霸昏。
