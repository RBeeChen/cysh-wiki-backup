「一起趣桌遊館」拒絕嘉中學生消費所引起的事件。

## 事件起源

辯論社學弟跟某友校一起去打桌遊，結果打到一半想吃飯，大家就出去吃，吃完大家就忘記要回桌遊店，然後就忘記付錢，結果被桌遊店老闆的女兒在臉書發文暗示。  
學弟們立馬知道自己忘記繳錢，便傳訊息道歉並願意支付未付的錢，結果老闆不領情表示不收，於是就宣稱拒絕與嘉中往來。
學弟打電話試過惹，也在店家樓下等了快一小時，但是老闆表示不願意原諒。

## 事件經過

[![](https://i.imgur.com/RQ7gpM4.png)](https://imgur.com/RQ7gpM4)

抵制嘉中學生文(備份截圖)

當日，桌遊店便Po文表示：

```
  即日起本店禁止嘉義高中的同學進                                                  
  謝謝^^
```

之後，該文經數十名網友分享、討論，並在「靠北嘉中」、PTT chi板等嘉中人平台引起數篇文的討論。  
同時，在該店家的粉專評價功能，也有數位嘉中人給予一星評價，並造成論戰。

最後該拒絕往來聲明文也遭店家自行刪除了
