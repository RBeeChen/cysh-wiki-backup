# 簡介

可以大膽地稱呼他為阿吉(某屆他導師班的學生稱呼他[阿吉仔])，開門見山地說，是位好老師（可是他姐姐說去當老師是沒用的人，所以常常調侃自己說自己是沒用的人）。

新北永和人現居後庄，出沒於嘉義市各咖啡店，嘉中最屌的國文老師之一(據說已經將&lt;師說&gt;背的滾瓜爛熟)，倡導窗戶不擦不然外面的陽光會刺傷人及地板不打蠟不然眼睛會受傷，擁有無限多的娃娃及玩具(據說他女兒是夾娃娃高手)，中秋節大概會帶全班去烤肉，非常大氣，常於班會課請學生吃點心或是贊助班級聚餐等。  
上課雖然看似瘋瘋癲癲（他寫的字可以去應徵美術老師了），卻蘊涵深不可測之實力，段考題目題型多變，富含創意，卻也相當具挑戰性！107及112學年曾讓一堆人國文不及格QQ(作文作業**一定要交**) 常自製各種阿吉精選國文詳解以及段考概要，自認如果自己第二帥就沒人敢說第一名，喇賽卻有人生的道理，但段考範圍的課文常常教不完，上課進度可能於一節課中從0到100。擅長於作文教學、猜題目及出題目，有時候認為段考題目太爛的話會不想檢討。

切記「不要跟他打嘴炮，因為你嘴不贏」，不然只要有機會，他就會嘴你，雖然嘴上說不在意成績但其實還是很在意(學長真心話:不要擺爛)，常強調自己是理性的人，同時又說愈強調自己理性的人愈感性。

高三會幫同學改備審，你花多少努力準備，他就會花多少精力幫你修改。

他作文都亂改，要求很多、價值觀很有問題（要有心理準備）

# 教學理念

「自由自在，自生自滅」，倡導學生的自主學習能力，所以平常上課玩手機或做其他事都不會管你，但其實他都看在眼裡，常常「看破不說破」，所以不要做壞事以為他不知道。

常常鼓勵學生，無論未來想做什麼
做自己就好，當想做的大人

# 趣事

1.擅長用閩南語唱李白的春夜宴從弟桃花園序

2.放學後如果在校園遊蕩就是忘記自己車停在哪

3.曾經在一個月內丟掉兩次手機

4.宿醉來上課卻沒人看得出來，(無異樣，完全不影響教學能力)

5.上課走動經過戴著眼鏡睡覺的同學身旁(輕拍)微俯身道：「睡覺不要戴眼鏡」接著繼續若無其事的唸課文

6.喜歡大雅牛排的玉米濃湯

7.曾經想成為史上聯考國文滿分分第一人，選擇滿分但敗於作文，因為本人表示選擇題很仔細研讀作答
選擇題全對，結果作文寫到一半鐘響，時間管控不佳班上還有兩名同學比他高

8.情緒高昂時會跌坐在講台上

9.擁有許多筆名(可以從他撰寫的講義中尋找)

10.在某班曾因太激動肘擊黑板而將上面的時鐘擊落

11.表示自己會抽菸但不會在學校抽菸，曾多次表示大學多次翹課跑到外面抽煙

12. 曾經到8:20還沒入班上課，根據他的說法，教務處通知他的時候他正在學校游泳池游泳，結果嚇一個馬上跳上岸趕快衝來上課（發生於2025年

13.口頭禪是「好不好」，幾乎講完任何話後面都要加一句 好不好

14.當年為中興大學校史第一人考取師大國文研究所
而且是榜首之姿，但當年因為大學有一門課不及格而沒辦法畢業（本人表示常常翹課抽菸），搞到後來該名教授在學校壓力之下硬是給他及格，
（因為學士班沒畢業無法繼續讀碩士
