## 學歷

國立臺灣師範大學資訊教育研究所碩士(1995)

## 有趣的小知識

1.以前都叫他的教授是老闆

2.他有一個表弟的小孩要請教他要填什麼科系,他要他填最難的科系就對了,目前已經快考到會計師執照了

3.常常講話時講到一半忍不住打哈欠,因此聲音會變得很奇怪

4.英文發音特殊

```
 true==Troo  

 judge==Judige  

 distriction==dis!@#!~%tion  

 Castle==K死偷  

 Zombie==loombi
```

5.聽說他大學時很愛騎車耍帥,也雷過很多次田  
然後有次被車撞飛了出去,好像頗遠頗高,但因為有安全帽讓他安然無恙  
結論:騎車一定要戴安全帽,還有品杰在撞飛那時並沒有人生的跑馬燈出現只出現鏗鏗鏘鏘的聲音

6.很認真的老師  
[資訊研究社](資訊研究社 "資訊研究社")指導老師  
看起來非常不友善，但實際上相處過後真的是個大好人  
不論多元選修是否滿額，有學生要轉去他的班他絕對極力幫忙  
如果有學弟沒搶到他的課可以自己跑去和他問看看  
他強調:期中考的用意只是想檢測最基本的東西，他自己也懶得改補考考卷  
所以上機考都考教過單元的前三題(爆簡單)  
如果跟著老師好好學肯定會收穫很多  
不想認真學程式也不會太難拿到學分  
每次上機考前會先打開考試作弊處罰來警告學生  
壓力好像很大開玩笑都是地獄梗的等級  
很愛面子

7.上資訊概論的時候會到 pchome 找天價的電子產品說：「這不貴，我家好幾個」。  
上課曾經跟同學推銷自己的防蚊液。

8.面貌與國立嘉義大學林宸堂教授相似

9.名言:  
這很直覺嘛  
講過N百遍的東西  
國小到底在幹嘛連打字都不會  
別作弊被抓到然後被我記過之後再來那邊哭哭啼啼   
這題這樣寫就臭掉了

10.每次上機考(現幾乎沒有了)前會先打開考試作弊處罰來警告學生

## 重點

上他的課不能睡覺，不能玩手機，也不能吃東西
