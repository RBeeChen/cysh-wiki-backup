橘色: 站起來做望遠動作

#### 旋轉木馬

紅色: 被抽到(站起來並上下慢速晃動)  
橘色: 站起來轉圈

抓: 最慢or做錯or站錯的人

[![](https://i.imgur.com/m07uAgF.png)](https://imgur.com/m07uAgF)

#### 神槍手

**特殊指令**: 神槍手 n  
n = 執行次數  
紅色: 被抽到(站起來叫其他人名字) **千萬不要叫黃建瑜，不然你會被點一整節**  
綠色: 被叫到名字(站起來點其他人(一定要講名字))  
若站起來的人數等於n+1的時候就要立刻坐下

抓: 叫人停頓太久or最後坐下的人  
以下演示「神槍手 3」:

[![](https://i.imgur.com/fSdXZuQ.png)](https://imgur.com/fSdXZuQ)

#### 迴旋鏢

**特殊指令**: 迴旋鏢 n  
n = 執行次數  
紅色: 被抽到(站起來叫其他人名字) **千萬不要叫黃建瑜，不然你會被點一整節**  
綠色: 被叫到名字(站起來點其他人(一定要講名字))  
若站起來的人數等於n的時候就要點被抽到籤人的名字，點完後立刻坐下

抓: 叫人停頓太久or最後沒有點被抽到籤人的名字的人or最後坐下的人  
以下演示「迴旋鏢 4」:

[![](https://i.imgur.com/0C8rtc9.png)](https://imgur.com/0C8rtc9)

綠色: 被指到(站起來指向前後左右沒有站起來的人)

抓: 沒有方向可以指的人(淺綠色)
