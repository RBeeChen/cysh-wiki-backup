# 簡介

高一高二稱**科展研究社**，高三稱**個別科學研究社**，因條目過於相似，故放在同一個條目中。  
性質相似於以往的[高瞻三社](高瞻三社 "高瞻三社")，如果你是[科學班](科學班 "科學班")的學生，那麼高二開始就可以無條件加入此社團，而非科學班的學生就相當遺憾，無法加入。本社團不會全部都是死板的科學課程，而是藉由與社團成員之間的合作，研究一系列實驗，做出實驗報告，參與科展競賽，每年暑假前後會邀請地區知名國中與友校參與成果發表會，是一個可以豐富學習歷程的社團之一。  
有時候社團在高一下中出現奇奇怪怪的名稱，就跟[三年級](三年級 "三年級")的社團一樣。

# 創始

據某退休老師所述，在科學班成立之前，[數資班](數資班 "數資班")和[高瞻班](高瞻班 "高瞻班")是強迫要做獨立研究的，在[科學班](科學班 "科學班")成立之後，前面所述的述的兩個班改成自由參加，後者採強迫參加(因為獨立研究是科學班學程的一部分)。直至第十屆科學班，高二才開始赴大學做獨立研究，從第十一屆至第十三屆，高一下就會開始做獨立研究，提早準備一年後要參賽的科展作品。據說第十四屆是可以自由選社兩學期的。

# 個別科學研究

高三過資格考的科學班學生，必須做個別科學研究始得畢業，題目不得與之前自己做的題目相同。

# 優點

* 可以增加學生們對自然科學理論的熟悉
* 可以讓學生參與科展競賽，增加得獎機會
* 社團風氣是全部社團中最上進的
* 別人在學校上社課，科學研究社的學生可以去大學研究科展
* 可以不用支出鉅額社費就可以享有充沛的國家資源(據說一屆約三百萬)
* 有至少四分之一的社員是女生
* 可以體驗碩士生活
* (據說)不會參加社團評鑑
* 可以豐富學習歷程
