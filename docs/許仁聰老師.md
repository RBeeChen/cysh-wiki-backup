## 簡介

現任職補教界與興華中學(在王宇集團收購興華中學後)，又名&lt;許人聰&gt; (為補習班藝名)、&lt;愛因斯聰&gt;  
教學風趣認真  
民國36年出生
第一次離開嘉中據說和[李文堂老師](/index.php/%E6%9D%8E%E6%96%87%E5%A0%82%E8%80%81%E5%B8%AB "李文堂老師")有關

之前任職於嘉中時被校長認為是問題老師,有留過長髮,被批評沒有老師樣,之後剪成光頭,但因為習慣於長髮而常常甩頭想把頭髮甩到後面,人家還以為他中風
在他帶的班級曾經實施午休讓全體躺著睡覺（當然地每天拖）。擅長打羽球，常在嘉義某羽球館出沒。曾強調四維空間的第四維即是「時間」。
目前補教界上課喜歡用節奏感的口語來幫助學生記憶,例如：

|---------♩--------♩--------♩-------- ♩|

```
    向右   看成   左
```

| ----------♩--------♩--------♩-------- ♩|

```
     減號   看成   加     號啊
```

| ----------♩--------♩--------♩-------- ♩|

```
     右     加    左
```

解題時，會發出「歐～～～～後」的聲音，然後強調這個解題法是屬於流星級（意即非常快速的解題方式）或者是「彗星級」（比流星級更為高級快速的方式）。
退休前即在仁愛路勵進補習班長期兼課，也常到台中水利大樓補習班上課，若是在學校所任教的班級可免費到補習班上課，雖然都免費給予一樣的教材，但在學校的授課內容遠不及在補習班的豐富且生動。

## 退休事件

傳聞本欲接任教務主任，但因為依「教育人員任用條例」第 34 條規定，「專任教育人員，除法令另有規定外，不得在外兼課或兼職。」似乎因而被檢舉而無法繼續留任，最終導致退休離去。
