# 特色

1. 七點半到校
2. 高二時會跟高三考第一次模擬考
3. 學校會投入大量資源在這些班級，所以一堆學生搶著報名
4. 各科會安排加考
5. 英文單字進度是普通班的兩倍
6. 有許多校外參訪的機會
7. 有自己獨立的進度
