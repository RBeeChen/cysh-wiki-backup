嘉義高中模型研究社 (CYSH Model Club)

為嘉中前三大技術社團，有良好的社團環境以及用心的社團老師，社辦在生活科技大樓四樓，窗外遠眺可以看到嘉商的操場和一旁的棒球場，如果棒球場有比賽是可以看到的，製作模型的基本工具社辦都會有，允許社員將未完成的模型放在社辦，減輕社員上課時攜帶工具的負擔。

社團的學習氛圍非常良好，社團老師會提供專業的指導和建議，幫助社員們克服困難和問題。在社團中，社員們可以發揮自己的創造力和想像力，通過製作模型來提高自己的動手能力和藝術素養。此外，學生們還可以在社團中結識到志同道合的朋友，共同學習和成長。

```
期末會有一節吃東西的時間(通常是最後一節)
為了鼓勵社員參與(順便做做業績)，所以有設立獎金(資格:只要你是現任社員)(達標方法:參加任何一個與模型相關的比賽，不管名次)(名額:一學期總共兩個)(金額:一個名額500新台幣)(領取方法:看當屆社長如何處理)

社費

```
  看當其公布的預算書(可以去學校的網站查詢) (鋼彈、軍模的材料費另算) 
  每學期繳一次，不包含社服費用(社服的部分就看當屆的幹部還有沒有心力實行了)
