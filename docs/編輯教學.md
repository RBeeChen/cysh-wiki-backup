## 一般編輯

語法和維基百科相同，如遇到問題可以點選編輯器上方的**說明**選項

語法完整使用請見[中文維基百科:使用說明:編輯頁面](https://zh.wikipedia.org/wiki/Help:%E7%BC%96%E8%BE%91%E9%A1%B5%E9%9D%A2)

## 插入圖片

僅支援來自[imgur](https://imgur.com/)圖床的圖片

語法：

```
 &lt;imgur&gt;XXXOOOO&lt;/imgur&gt;
 &lt;imgur float="right" width="600" comment="一些字"&gt;XXXOOOO&lt;/imgur&gt;
```

tag內放上imgur的ID，也就是網址上七碼左右的不規則字  
能用的參數有三個**float**, **width**, **comment**

* float：決定圖片要靠左(left)還是靠右(right)，預設靠右
* width：圖片寬度，單位px，預設為320
* comment：圖片下方的文字

[原始碼](https://github.com/ldkrsi/mediawiki_imgur)

## 插入影片

可以插入來自Youtube的影片

語法：

```
 &lt;youtube&gt;OOOXOXXO&lt;/youtube&gt;
 &lt;youtube width="320" height="190"&gt;OOOXOXXO&lt;/youtube&gt;
```

tag內放上Youtube的影片ID，也就是網址上v=之後11字左右的字串  
可用的參數有**width**、**height**，用來設定影片畫面大小，單位px，預設為640和385
