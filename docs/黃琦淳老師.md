## 簡介

講英文的口音很特別 會強調s，t 這些尾音  
用PPT上課 要練手速 否則抄不到筆記，  
2013年曾赴奧地利旅遊15天[[1]](#cite_note-1)  
待學生和善？？  
作風低調,行蹤隱密  
建議自讀 就當自修課  
甲狀腺抗進  
出考卷超級無敵簡單   
不用麥克風講話時比螞蟻還小聲，用了之後隔兩間教室都聽得很清楚   
堅持不用粉筆   
會說自己導班不可教化   
吃冰會叫你丟垃圾桶   
她說要退休了   
每一屆幾乎都帶1年1班

## 口頭禪

1. 有點給他
2. 把那些趴趴熊叫起來
3. 你們的心還是沒有靜下來
4. 像我都已經那麼瘦了
5. 那個麻煩同學上課不要......謝謝
6. 這個沒有問題(解題時)
7. 當仁………(當然)
8. 不要做瓜田李下的事情
9. 休眠中的叫起來

## 要求

1. 每天讀英文至少15分鐘 (過了一學期要變30分鐘)
2. 上課不要上廁所
3. 上課不要吃東西（討厭塑膠袋的聲音）
4. 上課不要跟同學聊天
5. 一天要背5個單字
