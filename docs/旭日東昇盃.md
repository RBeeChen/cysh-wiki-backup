# 參加辦法

6人一組報名參加，別擔心，升高二的學生與升高三的學生分開辦理。  
參賽選手會因為參加班級性質與學校等級分成一般組與資優組。

# 賽制

主要分為接力賽、個人賽、團體賽

## 接力賽

3人一小組，題目很簡單但很容易失手。(每題20分共6題滿分120分)

## 個人賽

題目非常艱深，通常都是嘉中[那些把學生整死的命題老師](/index.php/%E5%98%89%E7%BE%A9%E9%AB%98%E4%B8%AD%E8%B6%85%E5%8F%AF%E6%80%95%E5%87%BA%E9%A1%8C%E8%80%81%E5%B8%AB "嘉義高中超可怕出題老師")命題。影響到個人獎項的重要比序。(滿分50\*6=300分)

## 團體賽

也是仿TRML，簡單來說就是大家一起討論題目啦。(每題6分共20題滿分120分)

# 獎項

當天比完當天公布

## 個人獎項

金牌前10%

銀牌前10~30%

銅牌前30~60%

## 團體獎項

一般組獎項、資優組獎項、總獎項
