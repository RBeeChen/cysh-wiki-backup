# 技術紀載

## 圖書館電腦關閉還原

1. 到 C:\ProgramData\Microsoft\Windows\win10uwf，找到uwf\_disable\_only.cmd
2. 右鍵並選擇「以管理員身分執行」
3. 等待他重新啟動
4. 完成

這時你可以做任何變更

要儲存只要依照以下步驟開啟還原

1. 到 C:\ProgramData\Microsoft\Windows\win10uwf，找到uwf\_all\_enable.cmd
2. 右鍵並選擇「以管理員身分執行」
3. 等待他重新啟動
4. 完成
