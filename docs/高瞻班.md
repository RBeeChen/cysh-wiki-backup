## 簡介

100學年起甄選普通班新生成班，定位介於普通班和資優班之間  
  
最早起源於九零年代初期以能力分班的[對照班](/index.php?title=%E5%B0%8D%E7%85%A7%E7%8F%AD&action=edit&redlink=1 "對照班（頁面不存在）")，是早年升學主義掛帥時期的產物。  
以會考成績入學，通常錄取標準都在100分左右，不算太困難

高瞻計畫已終止，所以現在算是沒有錢的能力編班班級

本班之名--高瞻計畫班--於110學年度正式刪除，目前改稱\*15班經營
111學年正式更名為[高瞻雙語實驗班](/index.php/%E9%AB%98%E7%9E%BB%E9%9B%99%E8%AA%9E%E5%AF%A6%E9%A9%97%E7%8F%AD "高瞻雙語實驗班")

擁有自己的的高瞻桌球室（只是掛名而已，其實都是老師在打）
資優大樓一樓有高瞻實驗室（也只是掛名）現在歸李文堂老師管，也是TYPT基地

```
 同學：（我是誰？我在哪？）   

 p.s.聽說模擬考之類獎狀還是會寫高瞻班
```
