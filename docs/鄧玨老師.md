[鄧開玉老師](/index.php/%E9%84%A7%E9%96%8B%E7%8E%89%E8%80%81%E5%B8%AB "鄧開玉老師")的兒子

會違反教育部規定對沒穿運動服的學生登記週末輔導（也是體育組長規定才這麼做的）

於112下學期考上嘉中正式老師（扶正）

113學年度接任可悲行政（補校校務組長）
114學年度擔任班導 幾乎不在班上（看起來很忙）班上事務基本上都是班長副班長主持

與學生聊天表示自己又當組長又帶四個班超忙 四個班裡面還有雙語班更忙了

專長是網球，會和學生一起打籃球，是嘉中少數親近、顏值top的體育老師
單挑贏他的請飲料

會抓玩手機的（很多老師都會吧）

有個財富自由的朋友
