1. 高三大致上不能偷玩手機
2. 不能口出穢言
3. 不能趴著睡覺(可是允許以其它姿勢睡覺，例 : 撐著睡)
4. 要攜帶課本
5. 不能脫鞋子
6. 不能借用健教課做體育活動(例：打籃球、練習田徑)
