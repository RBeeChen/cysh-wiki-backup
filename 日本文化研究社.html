社團詳介​

: 一個很奇妙的社團，介於"玩樂"及"學技術"兩種社團取向之間，看似整天在ㄎ一ㄤ，但是又會讓你覺得好像又學到東西，每堂課的分貝理論上都會很高。

: 曾短暫擁有社屬台灣民俗技藝團

名稱​

: 日本文化研究社，簡稱"日研社"，常被外界戲稱"日顏射"，原本想叫"酷日本社"，但由於本名稱太過"酷"了被校方拒絕。

: 第三屆社長自稱為"大麻研究社"。

起源​

: 2017年時，由兩個瀕臨人數不足倒社危機的研究日文社團合併並轉型成的新生社團，改為以研究各種日本文化為主，研究日文為輔的社團，由當時就在嘉中任教日文多元選修課的王嘉暄老師的妹妹--ひとみ老師擔任初任社師，並莫名其妙地由黃伯溫學長擔任初代目。

: 在第三屆時，由於ひとみ老師要去追求人生幸福的原因改為由王嘉暄老師擔任社師，但他本人由於其他工作因素有時不會出席社課，也沒有太多的心力準備社課內容，第三屆的社課都是由幹部和老師討論主題過後，由老師提供器材、教具，幹部們規畫具體課程內容，社奴製作「幹他媽超機掰屌的超可動PPT」，再由社奴在社課時上去講課。

瓶罐​

初代目化​

: : 據說和**"初代目"相處許久的人有機會觸發"初代目化"的技能。此技能覺醒條件為跟初代目一起經歷約莫半年的修行過程，過程包含:PPT製作、PPT素材外拍、影片剪輯、台灣民俗傳統舞蹈、日本傳統舞蹈、使用日文演講、製作彈珠台等。 : 剛覺醒此技能時會因為在與初代目聊天時，對方傳出訊息之前就可以得知對方在想什麼而感到神奇。然而這只不過是初期而已，完全初代目化則可以解鎖腦袋同步**，甚至是身心靈超感知等特殊功能。 : 據說初代目化的威力甚至遠遠超越腥電桿硬(詳見 日研不思議-歷屆社長慣例)，然而至今只有叔叔一人成功初代目化。

被詛咒的職位​

: SCP-0087

項目等級​

: Thaumiel

特殊收容錯施​

: SCP-0087是發生在擔任█████職位的社員上的異常現象，造成SCP-0087發生的人員稱為SCP-0087-1，由於SCP-0087發生期間會影響除████職位的幹部一人外的其餘社員的腦波，使受影響著深信並認同SCP-0087-1的一切行為，深信其一切行為皆具有合理性，並且不會提出任何質疑。所以只能由████職位的幹部進行SCP-0087的影響處理並將其影響最小化，其影響不限現任的████職位的幹部，甚至會波及前幾任的████職位幹部。████職位的幹部必須具有極強的情感承受能力，與危機處理能力，並且要在不使用"物理"手段的情況下解決其影響，因此交接時要格外重視████職位的幹部的選擇。

: 由於SCP-0087-1職位的不可根除性，所以要徹底根除其對社團的影響幾乎是不可能的。

: 於2019/██/██開始，SCP-0087-1不可自行選擇該職位的交接人員，以避免未來的下一位SCP-0087-1對社團造成進一步傷害。

: 發生事件2-8-4之後，社團█████████研究組織正在研究是因為當上了該社的████職位會使任位者不自覺地進行SCP-0087現象，或是由於人為疏忽導致當上████職位的人都是██。

描述​

: SCP-0087-1會在發生SCP-0087現象時會做出許多不可考的行為異常，例如:提出██████、莫名進行████預報、試圖██████30██等各種無法理解的行為。

: 這些行為除了████職位的幹部一人之外的一切人員皆無法發現其怪異、無邏輯性甚至█████性，甚至會受到SCP-0087-1的洗腦萌生這些行為皆十分有趣、好玩且十分可行的想法，所以經常出現大多數職員贊同SCP-0087-1之行為，導致收容難度加劇。

: 目前已成為SCP-0087-1的人員名單:████、█████、████ [須獲得 日研社除█████職位外的幹部 才可解鎖閱讀權限] 等人，三人除了皆為亞裔黃種人、擁有中華民國國籍、曾就讀嘉義高中及曾任職日研社的████職位之外其他的共同點有待█████████研究組織的進一步研究。
